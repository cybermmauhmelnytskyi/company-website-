Практична робота: Симуляція реального ІТ-проєкту

Локальна робота

  • Ініціалізація репозиторію:

    git init

  • Перевірка статусу файлів:

    git status

  • Додавання файлів до індексу:

    git add .

  • Фіксація змін (комміт):

    git commit -m "опис"

Робота з гілками

  • Створення нової гілки:

    git branch name

  • Перехід на іншу гілку:

    git checkout name

  • Створення та перехід:

    git checkout -b name

  • Злиття гілок:

    git merge name

Робота з GitHub

  • Клонування репозиторію:

    git clone URL

  • Прив'язка віддаленого резу:

    git remote add origin

  • Завантаження змін з сервера:

    git pull origin main

  • Відправка змін на сервер:

    git push origin main
